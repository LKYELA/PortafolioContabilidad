## Responsive Breakpoint Summary

| Breakpoint        | Major Changes                                                                                                                                                                                                 |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `≤ 1024px` (tablet) | Nav gaps & font sizes increase via `clamp()`. Social icons still **visible**.                                                                                                                                  |
| `≤ 960px`          | `.hero-split` → stacks to single column; `.hero-profile` → horizontal row; calendar carousel → 2 cards per view (`≥ 1000px` exactly, 1 card ≤ 1000px).                                                       |
| `≤ 768px` (mobile) | Header `min-height: 60px`; `.social-icons` → `display: none`; `.menu-toggle` → visible; `.menu` → fixed full-width drawer; `.services-grid` → single column; `.about-content` → single column; `.footer-content` → single column. |
| `≤ 600px`          | `.hero-profile` → stacked (profile-name + profile-role column). Typography further reduced.                                                                                                                   |
| `≤ 480px` (small)  | Logo & menu link font sizes slightly tighter.                                                                                                                                                                |
